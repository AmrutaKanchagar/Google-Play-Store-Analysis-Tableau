# 📱 Google Play Store Analysis Dashboard (Tableau)

## 📌 Project Overview
This project presents an interactive Tableau dashboard that analyzes the Google Play Store dataset. It provides meaningful insights into app performance, user engagement, installation trends, ratings, reviews, Android version compatibility, pricing, and content categories through interactive visualizations.

---

## 🎯 Objectives
- Analyze app ratings and user reviews.
- Compare app installations based on app size.
- Explore Android version compatibility.
- Study content ratings and app genres.
- Compare free and paid applications.
- Present insights through interactive dashboards.

## 🚀 How to Run

1. Clone this repository.
2. Open **Google_Play_Store_Dashboard.twbx** using Tableau Desktop or Tableau Public.
3. Explore the interactive dashboard using filters and charts.

---

## 📌 Key Insights

- Most applications are free.
- Ratings generally range between 4.0 and 4.5.
- Certain categories dominate app installations.
- App size influences installation trends.
- Android version compatibility varies across applications.
- Reviews provide valuable insights into user engagement.
